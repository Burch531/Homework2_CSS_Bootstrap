About Me

Hi! I'm JAmie Burchell currently an Sr IT Analyst in a data warehouse for a healthcare company. Our corporate data warehouse deals with big data across multiple platforms. The main system that i am involved with is Oracle this is the database that most other platforms connect to to manage the data.I am able to write queries and create, update and delete statement into tables. Next system is SQL and we also handle a lot of our data flow and have access to write queries in these databases. Other system that I monitor are Informatic, Snaplogic, Attunity, Data Mirror and Netezza. I am responsible for hundreds of jobs across all of these systems. When a job fails I am in charge of finding the failure, pulling the code or data to figure out what the issue is and getting it corrected to successfully rerun the job.
